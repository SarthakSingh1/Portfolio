Landmark Road Trip Planner

This tool allows you to plan a road trip where you minimize driving time while maximizing the attractions you see on the way

Demo

Project Overview

This project uses a CSV of Attracktions and their Longitude and Latitude coordinates to create a undirected graph which is then traversed with a variety of graph traversals. It provides you with a path that prioritizes distance and another one which prioritizes landmarks.
