Conflux

Distributed stream processing engines like Apache Flink leverage parallelism to process streams of data incredibly quickly. The issue with paralleism, it messes up the ordering. The order in which stream are sent into flink are not how they leave. Conflux is a system which allows you to leverage Flink while maintaining order and beats existing solutions by a factor of 40%.
